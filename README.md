# README

* Ruby version: 2.6.0

To start the app locally:

        bundle install
        
        rake db:migrate
  
        rake db: seed
        
        rails s

The app has been deployed on heroku https://chronicle-of-higher-education.herokuapp.com/api/v1/orders

The end points are 
      
      /api/v1/menu 
      
      /api/v1/items
      
      /api/v1/orders
      
      /api/v1/orders/:id
